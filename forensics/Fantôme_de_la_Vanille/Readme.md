
Ça, c'est du base64 `WE9SKGZsYWcsIE1ENShBdXRob3JfbWV0YWRhdGEpKQ==`
Decryptons le!

```bash
$ base64 -d << EOF
WE9SKGZsYWcsIE1ENShBdXRob3JfbWV0YWRhdGEpKQ==
EOF

```

Résultat:
---
```text
XOR(flag, MD5(Author_metadata))

```

Une indice pour decrypt le payload!

Donc, ce payload a été crypté selon cette methode. Il faut donc le decrypt.

Nous avons l'Author_metadata = OCOI2026 et le payload.
> Cryptage md5
```bash
$ echo -n "OCOI2026" | md5sum

```

Résultat:
---
```text

```

Créons un script python!
```python3
import base64

payload = 'ymHNxXIjUXxF0TlCNoBCsLxOtuIkSlk5Es07WgWFc975Tra9LkpZOxbXfFM=h-'
md5_hex = '8922828c40152a0a71bf082e5ab41d81'

# On retire d'éventuels caractères parasites à la fin (comme le 'h-') pour le base64
clean_payload = payload.split('=')[0] + '=' * (len(payload_b64) % 4)
payload_bytes = base64.b64decode(payload.split('h-')[0])

key_bytes = bytes.fromhex(md5_hex)

result = ""
for i in range(len(payload_bytes)):
    char_xor = payload_bytes[i] ^ key_bytes[i % len(key_bytes)]
    result += chr(char_xor)

print(f"Le Flag est : {result}")
```
